A Cordova project with reactJS
===============================
First run

React 15.0.1 is not supported by
 - material-ui ( 0.15.0-alpha.2 )
 - react-tap-event-plugin ( 0.2.2 )

React 0.14.0 is required to use both material-ui & react-tap-event-plugin.


### Running Project
 `npm run 2js`
 `npm run pack`
